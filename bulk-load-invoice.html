Follow these steps to upload invoice
  1. Download the CSV template.
  2. Fill in or update the CSV file.
    • Fields marked with a "*" are mandatory.
    • Each row uploaded will create a new invoice.
    • Click Start Upload and the system will verify the file using the first 6 rows, and load the file if no errors are found.

  3. Load the updated file

    4. Note: If you are loading csv files with non-English characters, please consult the following help note.
