User Details

Roles

To create or manage expense reports, the user must have the right roles and marked as an active Expense User.

  • Guest » Implicit role for all accessors of the site
  • Admin » Full system access to setup and maintain the application
  • User » Standard role for all users who need to create and/or approve requisitions
  • Buyer » Adds ability to manage requisitions, orders, RFQs, suppliers and items, plus access to purchase order reports
  • Accounts Payable » Adds ability to manage suppliers, view orders and create invoices
